# lkr-website

Immersive single-page website concept for **Earth Alliance Nagaland** with:

- 3D animated hero background using Three.js
- Scroll-based reveal and motion effects using GSAP ScrollTrigger
- NGO-focused mission, impact, and engagement sections

## Run locally

No build step is required.

```bash
python3 -m http.server 8080
```

Then open `http://localhost:8080`.

## Notes

The requested command `uipro init --ai copilot` is not currently available in npm in this environment, so the website is implemented directly with static HTML/CSS/JS.
